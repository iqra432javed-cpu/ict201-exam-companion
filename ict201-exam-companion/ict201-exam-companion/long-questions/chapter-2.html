Q1.Discuss in detail the types of terminals and the various Source Data Automation (SDA) devices used in modern data entry.
10 Marks▼ show answer

Types of Terminals

A terminal is a device used to enter data into, and receive output from, a host computer system.

  • Dumb Terminal: Has no local processing power; merely displays input/output, fully dependent on the host.
  • Smart Terminal: Has limited local processing, such as validating or formatting data before transmission.
  • Intelligent Terminal: Contains its own CPU and memory, able to run programs and process data independently of the host.

Source Data Automation (SDA)

SDA captures data directly and electronically at its point of origin, eliminating manual transcription and reducing errors. Key SDA devices include:

  • Barcode readers — scan striped product codes at retail checkouts.
  • OCR (Optical Character Recognition) — reads printed text directly into digital form.
  • MICR (Magnetic Ink Character Recognition) — reads magnetic ink numbers on bank cheques.
  • OMR (Optical Mark Recognition) — detects pencil-marked bubbles, commonly used for exam answer sheets.

Together, terminals and SDA devices represent the two broad approaches to getting data into a system: manual/interactive entry through terminals, and automated capture at the source through SDA.

Q2.Discuss in detail the various output devices: monitors, printers, and plotters.
10 Marks▼ show answer

Monitors

Monitors display soft-copy output. CRT monitors use a bulky cathode-ray tube, consuming more power and space. LCD monitors are flat and energy-efficient, while LED monitors are LCDs with LED backlighting, offering better brightness and slimmer designs. Monitors are further classified by colour capability and resolution (the number of pixels displayed).

Printers

Impact printers (Dot Matrix, Daisy Wheel) physically strike an inked ribbon against paper — noisy, but cheap and able to produce carbon copies. Non-impact printers (Inkjet, Laser, Thermal) form output without physical contact — spraying ink, fusing toner with heat, or using heat-sensitive paper — producing quieter, higher-quality results.

Plotters

A plotter produces high-precision, line-based graphics by moving a pen continuously across paper, rather than building an image from a grid of dots. This makes plotters ideal for engineering drawings, architectural blueprints, and large-format maps, where smooth continuous lines matter more than dense colour or text output.

Q3.Discuss computer viruses in detail and explain the security measures used to protect against them.
10 Marks▼ show answer

Computer Viruses

A computer virus is a malicious program that attaches itself to legitimate files or programs and replicates itself, often corrupting data or disrupting normal operation. Common types include:

  • Boot-sector virus — infects the area of a disk used to start up the computer.
  • File-infector virus — attaches itself to executable program files.
  • Macro virus — hides inside the macro commands of documents (e.g. Word/Excel files).
  • Worm — self-replicates and spreads across networks without needing to attach to a host file.

Security Measures

  • Installing and regularly updating reputable antivirus software.
  • Using firewalls to monitor and block unauthorised network access.
  • Taking regular, scheduled backups so data can be recovered after an attack.
  • Promptly applying operating system and software security patches.
  • Practising safe habits — avoiding unknown email attachments, untrusted downloads, and unsafe websites.

A layered combination of these measures, rather than reliance on any single tool, offers the strongest practical protection.

Q4.Explain in detail primary and secondary memory, including the different types of RAM and the role of cache memory.
10 Marks▼ show answer

Primary vs Secondary Memory

Primary memory (RAM, ROM, Cache) works directly with the CPU, offering very high speed but mostly volatile storage — data is lost without power. Secondary memory (hard disks, optical disks, flash drives) is slower but non-volatile, providing permanent, large-capacity storage that survives a power-off.

Types of RAM

  • SRAM (Static RAM): Fast and expensive; does not need constant refreshing; used in cache memory.
  • DRAM (Dynamic RAM): Slower and cheaper; must be refreshed thousands of times per second; used as a computer's main RAM.
  • SDRAM (Synchronous DRAM): DRAM synchronised with the system clock for faster, more predictable performance — standard in modern PCs.

Cache Memory

Cache is a very small, very fast memory layer placed between the CPU and RAM. It stores frequently used data and instructions so the CPU rarely has to wait on the comparatively slower RAM. Cache is organised into levels — L1 (smallest, fastest, closest to the CPU core), L2, and L3 (progressively larger but slightly slower) — balancing speed against cost and chip space.

Q5.Discuss in detail the structure and working of a hard disk, along with diskettes and RAID technology.
10 Marks▼ show answer

Hard Disk Structure & Working

A hard disk stores data magnetically on one or more spinning circular platters. Each platter surface is divided into concentric tracks, and each track is further divided into sectors — the smallest directly addressable unit of storage. A read/write head, mounted on a moving arm, hovers just above the spinning platter and moves to the correct track/sector to read or write data. The combination of platter rotation speed and head movement speed determines overall disk performance.

Diskettes

Diskettes (floppy disks) were an early, low-capacity, portable magnetic storage medium, used to transfer small files between computers before the rise of USB drives and cloud storage. They are now obsolete but historically significant.

RAID

RAID (Redundant Array of Independent Disks) combines multiple physical disks into one logical storage unit. By striping (spreading data across disks) and/or mirroring (duplicating data on multiple disks), RAID improves either performance, reliability, or both — allowing data to survive even if a single disk in the array fails.

Q6.Explain optical and magnetic storage technologies, and discuss the different types of backup systems.
10 Marks▼ show answer

Optical Storage

CDs and DVDs store data as microscopic pits and flats on the disc surface. A laser beam reflects differently off these features, and a sensor interprets the resulting light pattern as binary data. DVDs hold considerably more data than CDs due to tighter track spacing and, often, multiple data layers.

Magnetic Storage

Magnetic tape stores data sequentially along a long magnetic strip. While slow for random access, it is extremely cost-effective per gigabyte, making it the traditional choice for backing up and archiving large volumes of data that are rarely accessed but must be retained.

Backup Systems

  • Full backup: Copies all data every time — simplest to restore, but slow and storage-heavy.
  • Incremental backup: Copies only data changed since the last backup of any type — fast and efficient, but restoration requires every incremental backup since the last full backup.
  • Differential backup: Copies all data changed since the last full backup — a balance between speed and ease of restoration.

Organisations typically combine a periodic full backup with frequent incremental or differential backups to balance speed, storage cost, and recovery reliability.

Q7.Explain the Data Communication Model in detail and discuss digital/analog transmission, modems, and the different transmission modes.
10 Marks▼ show answer

Data Communication Model

Data communication is the exchange of data between two devices through a transmission medium, governed by shared rules (protocols). The basic model has five elements: Source (generates data) → Transmitter (converts data into a signal) → Transmission System (the medium carrying the signal) → Receiver (converts the signal back into data) → Destination (uses the delivered data).

Digital vs Analog Transmission

Digital transmission sends data as discrete pulses (0s and 1s), offering high resistance to noise and is standard in modern networks. Analog transmission sends data as a continuously varying wave, historically used on telephone lines, but more prone to signal degradation over distance.

Modems

A modem ("modulator-demodulator") converts a computer's digital signals into analog form for transmission over telephone lines, and converts incoming analog signals back into digital data — essential for early dial-up internet access.

Transmission Modes

Asynchronous transmission frames each character individually with start/stop bits and needs no shared clock; synchronous transmission uses a shared timing clock for continuous, more efficient bulk transfer.

By direction: Simplex sends data one way only (television broadcast); Half Duplex sends data both ways but one direction at a time (walkie-talkie); Full Duplex sends data both ways simultaneously (a telephone call).
