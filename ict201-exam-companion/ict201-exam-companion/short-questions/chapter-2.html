Chapter 2 — Short Questions (5 Marks)

17 most important short questions on I/O Devices, Storage & Data Communication.

Q1.Differentiate between Dumb, Smart, and Intelligent terminals.
5 Marks▼ show answer
  • Dumb terminal: No local processing; entirely dependent on the host computer.
  • Smart terminal: Limited local processing, such as basic data validation before sending.
  • Intelligent terminal: Has its own CPU and memory, capable of running programs independently.
Q2.What is Source Data Automation (SDA)? Name two SDA devices.
5 Marks▼ show answer

Source Data Automation captures data directly and electronically at its point of origin, avoiding manual keyboard entry and reducing transcription errors.

Examples: Barcode readers (scan product codes) and MICR readers (read magnetic ink numbers on bank cheques).

Q3.Differentiate between soft copy and hard copy output.
5 Marks▼ show answer

Soft copy is a temporary, intangible output displayed on a screen or through speakers — it disappears once the device is switched off.

Hard copy is a permanent, tangible output, such as a printed document, which can be physically stored and handled.

Q4.Differentiate between impact and non-impact printers.
5 Marks▼ show answer

Impact printers (e.g. Dot Matrix, Daisy Wheel) physically strike an inked ribbon against the paper to form characters — noisy but cheap, and able to produce carbon copies.

Non-impact printers (e.g. Inkjet, Laser, Thermal) form output without any physical striking — by spraying ink, fusing toner with heat, or using heat-sensitive paper — producing quieter, higher-quality output.

Q5.What is a plotter? State one of its uses.
5 Marks▼ show answer

A plotter is an output device that draws images using continuous lines via a moving pen, rather than building an image from dots like a printer.

Use: Producing high-precision engineering drawings, architectural blueprints, and large-scale maps.

Q6.What is a computer virus? Name two of its types.
5 Marks▼ show answer

A computer virus is a malicious program that attaches itself to legitimate files and replicates itself, often damaging data or disrupting normal operation.

Types: Macro virus (hidden in document macros) and Worm (spreads across networks without needing a host file).

Q7.List any four computer security measures.
5 Marks▼ show answer
  • Installing and regularly updating antivirus software.
  • Using firewalls to block unauthorised access.
  • Taking regular data backups.
  • Applying software security patches/updates promptly and avoiding unknown email attachments.
Q8.Differentiate between primary and secondary memory.
5 Marks▼ show answer

Primary memory (RAM, ROM, Cache) is very fast and works directly with the CPU, but is mostly volatile — losing its contents when the power is switched off.

Secondary memory (hard disk, optical disk, flash drive) is slower but non-volatile, providing permanent, large-capacity storage that survives a power-off.

Q9.Differentiate between SRAM and DRAM.
5 Marks▼ show answer

SRAM (Static RAM) is faster and more expensive, and does not need to be constantly refreshed — commonly used for cache memory.

DRAM (Dynamic RAM) is slower and cheaper, but must be refreshed thousands of times per second to retain its data — commonly used as a computer's main RAM.

Q10.What is cache memory? Why is it important?
5 Marks▼ show answer

Cache memory is a very small, very fast block of memory placed between the CPU and RAM, holding frequently used data and instructions.

Importance: Since cache is much faster than RAM, it greatly reduces the time the CPU spends waiting for data, improving overall processing speed.

Q11.Define "track" and "sector" on a hard disk.
5 Marks▼ show answer

A track is one of many concentric circular paths on a hard disk platter where data is stored.

A sector is a smaller division within a track — the smallest unit of storage the disk's read/write head can address directly.

Q12.What is RAID? Why is it used?
5 Marks▼ show answer

RAID (Redundant Array of Independent Disks) combines multiple physical hard disks into a single logical storage unit using techniques like striping (spreading data) and mirroring (duplicating data).

Purpose: To improve performance and/or reliability, ensuring data survives even if one disk in the array fails.

Q13.How does an optical disk (CD/DVD) store and read data?
5 Marks▼ show answer

Optical disks store data as microscopic pits and flats on their surface. A laser beam reflects differently off pits versus flats, and a sensor interprets these differences in reflected light as binary data (0s and 1s).

Q14.Differentiate between full backup and incremental backup.
5 Marks▼ show answer

A full backup copies all data every time it runs — simplest to restore from, but slow and storage-heavy.

An incremental backup copies only the data changed since the last backup of any type — much faster and more storage-efficient, but requires all prior backups to fully restore.

Q15.List the five elements of the basic Data Communication Model.
5 Marks▼ show answer
  1. Source — generates the data.
  2. Transmitter — converts data into a transmittable signal.
  3. Transmission System — the medium carrying the signal.
  4. Receiver — converts the signal back into usable data.
  5. Destination — receives the delivered data.
Q16.Differentiate between digital and analog data transmission.
5 Marks▼ show answer

Digital transmission sends data as discrete electrical pulses (0s and 1s) — fast and highly resistant to noise.

Analog transmission sends data as a continuously varying wave — used by traditional telephone lines, and more prone to signal degradation over distance.

Q17.Differentiate between Simplex, Half Duplex, and Full Duplex transmission with examples.
5 Marks▼ show answer
  • Simplex: Data flows in one direction only. Example: television broadcast.
  • Half Duplex: Data flows both ways, but only one direction at a time. Example: walkie-talkie.
  • Full Duplex: Data flows in both directions simultaneously. Example: a telephone call.
