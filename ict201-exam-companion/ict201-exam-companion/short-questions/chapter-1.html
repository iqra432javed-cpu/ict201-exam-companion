Q1.Define a computer and name its basic elements.
5 Marks▼ show answer

A computer is an electronic device that accepts raw data as input, processes it according to a set of instructions, and produces meaningful information as output, while storing data for future use.

Basic Elements

  • Input — devices/methods that feed data in (keyboard, mouse).
  • Processor — the CPU, which executes instructions.
  • Memory — primary and secondary storage for data/instructions.
  • Output — devices that present processed results (monitor, printer).

These elements work together in a continuous cycle: Input → Process → Output.

Q2.Name and briefly describe the four generations of computers.
5 Marks▼ show answer
  • First Generation (1940s–50s): Used vacuum tubes; large, hot, and unreliable.
  • Second Generation (1950s–60s): Used transistors; smaller, faster, more reliable.
  • Third Generation (1960s–70s): Used Integrated Circuits (ICs); introduced high-level languages.
  • Fourth Generation (1970s–present): Used microprocessors; gave rise to personal computers.

Each generation reduced size and cost while increasing speed and reliability over the previous one.

Q3.Differentiate between hardware and software.
5 Marks▼ show answer

Hardware refers to the physical, tangible components of a computer — the CPU, monitor, keyboard, and printer — which can be seen and touched. Hardware can wear out or break physically.

Software is the intangible set of instructions (programs) that tells the hardware what to do, such as an operating system or an application. Software cannot be touched, only stored on a medium, and instead of wearing out it may develop logical errors called "bugs". Hardware and software depend on each other: hardware needs software's instructions, and software needs hardware to run on.

Q4.Differentiate between system software and application software.
5 Marks▼ show answer

System software manages and controls the computer's own hardware and resources so other programs can run — e.g. the Operating System, device drivers, and utility programs. It runs in the background and is essential to the machine's basic operation.

Application software is built for the end-user to carry out specific tasks, such as word processing, spreadsheets, or browsing — e.g. MS Word, Excel, or a web browser. While system software keeps the machine running, application software helps the user get real work done on top of it.

Q5.What is a Computer Based Information System (CBIS)? List its components.
5 Marks▼ show answer

A CBIS uses computer technology to carry out an organisation's information processing, converting raw data into useful information.

Five Components

  • Hardware — physical processing devices
  • Software — programs that direct the hardware
  • Data — raw facts to be processed
  • People — users and IT staff
  • Procedures — rules for correct system use
Q6.State the uses and limitations of application software.
5 Marks▼ show answer

Uses

  • Automates everyday tasks like typing, calculations, and presentations.
  • Supports communication, research, and data analysis.
  • Provides creative and entertainment tools.

Limitations

  • Depends entirely on compatible system software and hardware.
  • General-purpose packages may not fit highly specialised needs.
  • Requires regular updates and can carry security vulnerabilities.
Q7.Define the Arithmetic & Logic Unit (ALU) and the Control Unit.
5 Marks▼ show answer

The ALU is the part of the CPU that performs all arithmetic operations (addition, subtraction) and logical comparisons (equal to, greater than, less than) on data.

The Control Unit directs and coordinates the activities of the entire CPU. It fetches instructions from memory, decodes them, and directs the ALU and other components on how and when to execute them — essentially acting as the CPU's manager.

Q8.Briefly explain the importance of computers in modern life.
5 Marks▼ show answer

Computers are valued for their speed (executing millions of instructions per second), accuracy (negligible error when correctly programmed), large storage capacity, and ability to perform repetitive tasks without fatigue. They have transformed nearly every field — education (e-learning), business (record-keeping, automation), healthcare (diagnostics, patient records), and communication (instant messaging, video calls) — making processes faster, cheaper, and more reliable than manual methods.

Q9.What are the main limitations of computers?
5 Marks▼ show answer
  • No judgement: Computers cannot use common sense; they strictly follow instructions ("Garbage In, Garbage Out").
  • Dependency: They need electricity and correct programming to function at all.
  • Cost: High-end systems can be expensive to purchase and maintain.
  • Security risks: Vulnerable to viruses, hacking, and data loss if not protected.
Q10.Differentiate between a Mainframe computer and a Microcomputer.
5 Marks▼ show answer

A Mainframe is a large, powerful computer designed to serve hundreds or thousands of users simultaneously, commonly used by banks and airlines for mission-critical, high-volume processing.

A Microcomputer is a small, single-user machine built around a single microprocessor chip — the personal computers and laptops used in homes and offices. The mainframe vastly outperforms the microcomputer in processing power, simultaneous users, and cost, while the microcomputer wins on affordability and portability.

Q11.What is a Supercomputer? Give an example of its use.
5 Marks▼ show answer

A Supercomputer is the fastest and most powerful category of computer, capable of massively parallel processing — performing trillions of calculations per second.

Example use: Supercomputers are used for weather forecasting, nuclear and climate research, and complex scientific simulations that require processing enormous volumes of data far beyond the capability of ordinary computers.

Q12.Define Batch Processing and Real-time Processing with examples.
5 Marks▼ show answer

Batch Processing: Data is collected over a period of time and processed together at a scheduled point. Example: monthly payroll processing.

Real-time Processing: Data is processed immediately, as soon as it is entered, with results available instantly. Example: withdrawing cash from an ATM, where the account balance updates the moment the transaction occurs.

Q13.What is Source Data Automation (SDA)?
5 Marks▼ show answer

Source Data Automation is a method of capturing data directly from its point of origin using electronic devices, rather than manually typing it in. Common SDA devices include barcode scanners, OCR (Optical Character Recognition) readers, and MICR (used on bank cheques). Because the data is captured automatically at the source, SDA greatly reduces transcription errors and speeds up data entry compared to manual keyboard entry.

Q14.Define a Centralized Computing Facility.
5 Marks▼ show answer

A Centralized Computing Facility keeps all processing power and data on one large central computer, with users connecting to it through terminals. This model is easy to control, secure, and maintain since everything is managed from a single point — but it creates a single point of failure: if the central computer goes down, the entire organisation's processing stops, and heavy demand can bottleneck performance.

Q15.Define a Distributed Computing Facility.
5 Marks▼ show answer

A Distributed Computing Facility spreads processing power across multiple connected computers, often located in different places, which cooperate and share data as a single coordinated system. This model is flexible and resilient — if one node fails, others can continue working — but it is harder to coordinate, synchronise, and secure than a single centralized system.

Q16.Define a Decentralized Computing Facility.
5 Marks▼ show answer

A Decentralized Computing Facility gives each department or location its own independent computer system, with little or no coordination from a central authority. This gives maximum local control and faster local decision-making, but often leads to duplicated effort, inconsistent data across departments, and difficulty consolidating organisation-wide information.

Q17.What is the role of primary memory in a computer system?
5 Marks▼ show answer

Primary memory (mainly RAM) temporarily holds the data and instructions the CPU is actively working on at any given moment, allowing for extremely fast access compared to secondary storage. It is essential to processing speed because the CPU works directly with primary memory rather than the much slower hard disk. Most primary memory (RAM) is volatile — its contents are lost when power is switched off — while ROM, a non-volatile type of primary memory, retains essential start-up instructions permanently.
