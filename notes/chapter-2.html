1. Keyboard & Terminal Types

The keyboard remains the most common input device. Variants include the standard QWERTY layout, ergonomic (split/curved) keyboards designed to reduce strain, and on-screen virtual keyboards on touch devices.

A terminal is a device used to enter data into, and receive output from, a computer system — historically common with centralized facilities.

Terminal TypeProcessing AbilityTypical Use
Dumb TerminalNone — only displays input/outputPure data entry, entirely dependent on the host computer
Smart TerminalLimited local processing/editingCan validate or format data before sending it
Intelligent TerminalFull local processing (its own CPU/memory)Can run programs independently, e.g. a modern PC acting as a terminal

2. Dedicated Data Entry, SDA, Pointing Devices & Voice Input

Dedicated Data Entry (DDE): Data is first typed and stored on a separate medium (historically key-to-tape or key-to-disk systems) before being transferred in bulk to the main computer — separating data entry from processing.

Source Data Automation (SDA): Captures data directly and electronically at its point of origin, removing the manual typing step entirely. Common SDA technologies include:

  • Barcode readers — scan striped patterns on products.
  • OCR (Optical Character Recognition) — reads printed text directly.
  • MICR (Magnetic Ink Character Recognition) — reads the magnetic ink numbers on bank cheques.
  • OMR (Optical Mark Recognition) — reads pencil-marked bubbles, e.g. exam answer sheets.

Pointing devices let users interact with on-screen elements: mouse, trackball, touchpad, joystick, stylus, and touchscreen.

Voice input uses speech-recognition software to convert spoken words directly into digital data or commands.

Examiner's FavouriteQuestions frequently ask you to "differentiate SDA from manual keyboard entry" — always anchor your answer on accuracy and speed gained by skipping manual transcription.

3. Output Devices & Monitors

Output can be a soft copy — a temporary, intangible display (on a monitor or speaker) — or a hard copy — a permanent, tangible output (a printed page).

Monitors display soft-copy output. Types include CRT (older, bulky cathode-ray tube), LCD (Liquid Crystal Display, flat and energy-efficient), and LED (LCD with LED backlighting, brighter and slimmer). Monitors are also classified as monochrome or colour, and rated by resolution (the number of pixels they can display).

A plotter is a specialised output device that draws high-precision line-based graphics (engineering drawings, architectural blueprints, large maps) by moving a pen across paper, rather than building an image from a grid of dots like a printer.

5. Computer Virus & Security

A computer virus is a malicious program that attaches itself to legitimate files or programs and replicates itself, often damaging data or disrupting normal operation. Common categories include boot-sector viruses, file-infector viruses, macro viruses (hidden inside document macros), and worms (which self-spread across networks without needing a host file).

Security measures include installing updated antivirus software, using firewalls, taking regular backups, applying software patches/updates promptly, and practising safe habits such as avoiding unknown email attachments and untrusted downloads.

6. Storage Units: Primary & Secondary Memory

Data is measured in bits (binary digit, 0 or 1) and bytes (8 bits), scaling up through Kilobyte (KB) → Megabyte (MB) → Gigabyte (GB) → Terabyte (TB).

Memory TypeSpeedVolatilityExample
Primary MemoryVery fastMostly volatile (RAM)RAM, ROM, Cache
Secondary MemorySlowerNon-volatileHard disk, optical disk, flash drive
Diagram · The Memory Hierarchy
Top to bottom: speed and cost per byte fall sharply, while capacity rises sharply.

Primary memory works directly with the CPU for active tasks; secondary memory provides permanent, large-capacity storage that survives a power-off.

7. RAM Types & Cache Memory

  • SRAM (Static RAM): Faster and more expensive; used in cache memory; retains data without needing constant refreshing as long as power is supplied.
  • DRAM (Dynamic RAM): Slower, cheaper, and needs to be refreshed thousands of times per second; used as a computer's main RAM.
  • SDRAM (Synchronous DRAM): DRAM synchronised with the system clock for faster, more predictable performance — used in modern PCs.

Cache memory is a very small, very fast block of memory placed between the CPU and RAM, storing frequently used data/instructions so the CPU doesn't have to wait on slower RAM. It is organised in levels — L1 (smallest, fastest, closest to the CPU core), L2, and L3 (progressively larger but slightly slower).

8. Hard Disks, Diskettes & RAID

A hard disk stores data magnetically on one or more spinning circular platters. Each platter is divided into concentric tracks, and each track into sectors. A read/write head moves across the spinning platter to access or store data at the correct track and sector — the closer together these movements happen, the faster the disk performs.

Diagram · Hard Disk Structure
The head jumps between tracks; within a track, data is read sector by sector.

A diskette (floppy disk) was an early, low-capacity, portable magnetic storage medium, now obsolete but historically important for transferring small files.

RAID (Redundant Array of Independent Disks) combines multiple physical hard disks into one logical unit to improve performance, reliability, or both — by spreading (striping) data across disks and/or duplicating (mirroring) it, so that data survives even if one disk fails.

9. Optical & Magnetic Storage, Backup Systems

Optical disk storage (CD-ROM, DVD) stores data as microscopic pits and flats on the disk surface, read by a laser beam that detects changes in reflected light. DVDs hold significantly more data than CDs due to a tighter track spacing and, often, multiple data layers.

Magnetic tape stores data sequentially on a long magnetic strip — slow to access randomly, but extremely cost-effective for storing huge volumes of data, making it a traditional choice for backups and archives.

Backup Systems

  • Full backup: Copies all data every time — simplest to restore from, but slow and storage-heavy.
  • Incremental backup: Copies only data changed since the last backup (of any type) — fast and storage-efficient.
  • Differential backup: Copies all data changed since the last full backup — a middle ground between the two.

10. Data Communication & the Communication Model

Data communication is the exchange of data between two devices via a transmission medium, governed by agreed rules (protocols).

The basic Data Communication Model involves five elements:

  1. Source — generates the data to be sent.
  2. Transmitter — converts data into a transmittable signal.
  3. Transmission System — the medium that carries the signal (cable/wireless).
  4. Receiver — converts the incoming signal back into usable data.

Every data exchange — an email, a phone call, a web request — follows this same five-step path.

11. Digital & Analog Transmission, Modems

Digital transmission sends data as discrete electrical pulses (0s and 1s) — fast and highly resistant to noise/interference, used by modern computer networks.

Analog transmission sends data as a continuously varying wave — used historically by traditional telephone lines, and more prone to signal degradation over distance.

A Modem ("modulator-demodulator") converts a computer's digital signals into analog signals for transmission over telephone lines, and converts incoming analog signals back into digital form — essential for early dial-up internet connections.
